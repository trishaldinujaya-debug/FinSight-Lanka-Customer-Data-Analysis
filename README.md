# FinSight Lanka Customer Analytics Project

## Overview
This project analyzes customer savings behavior, loan portfolio risk, digital engagement, and cross-sell opportunities for a fintech company using Python, Power BI, and business intelligence techniques.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Google Colab
- Power BI
- PowerPoint

## Key Areas Analyzed
- Customer savings behavior
- Loan penetration and default risk
- Debt-to-savings analysis
- Digital engagement
- Cross-sell opportunities
- KPI dashboard creation

## Dashboard
Interactive Power BI dashboard developed to visualize KPIs and executive insights.

## Business Insights
- Mobile app users showed stronger savings behavior
- Certain loan segments exhibited higher default risk
- Cross-sell opportunities identified among savings-only customers
